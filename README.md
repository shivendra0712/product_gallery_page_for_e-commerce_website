# E-commerce Product Gallery Page

This repository contains the HTML and CSS files for a product gallery page designed for an e-commerce website, specifically showcasing footwear (Nike inspired).

## Purpose

The purpose of this page is to display a curated selection of products, categorized as "The Latest" and "Classics Spotlight," to encourage browsing and potential purchases. It features a clean layout with product images and basic navigation elements.

## Features

* **Navigation Bar:** Includes a logo (Nike placeholder), links to product categories ("New & Featured," "Men," "Women," "Kids," "Sale"), and utility icons (search, user, wishlist, shopping bag).
* **"The Latest" Section:** Displays a horizontal gallery of the newest products.
* **"Classics Spotlight" Section:** Shows a wider horizontal gallery of classic or highlighted products.
* **Image Galleries:** Uses `div` elements to organize product images in a row.
* **Basic Styling:** Provides a foundational visual structure using CSS.
* **Remix Icons:** Integrates Remix Icons for search, user, heart, and shopping bag icons.


## Technologies Used

* HTML5
* CSS3
* Google Fonts (Inter, Margarine, Parkinsans, Protest Revolution - imported but system-ui is used in CSS)
* Remix Icons (via CDN)
